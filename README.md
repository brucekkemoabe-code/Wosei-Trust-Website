# WOSEI Trust

Welcome to the official repository for the WOSEI Trust website. 

## About WOSEI Trust
WOSEI Trust is a non-profit organization dedicated to empowering communities and fostering sustainable development. The trust focuses on various initiatives aimed at uplifting local populations, providing resources, and creating opportunities for growth.

## About the Website
This repository contains the source code for the WOSEI Trust website. The site is built using standard web technologies:
- HTML5
- CSS3
- JavaScript

It features information about the trust, its beneficiaries, donation options (including mobile payments via Orange, Mascom, and beMobile), and news/updates.

## Local Development
To run this project locally:
1. Clone this repository.
2. Open `index.html` in your web browser.
3. Alternatively, you can run a local server (e.g., `python -m http.server`) from the project root.

## Deployment
This website is deployed using GitHub Pages.
